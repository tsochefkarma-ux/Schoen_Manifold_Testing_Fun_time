# Cell 34 — One-sided null relation and 15-dimensional HSS quotient

## Purpose

Cell 33 found that all three one-sided character evaluations have rank `5/6`,
and every symmetric character sector factors through rank `15/21`. Cell 34
extracts the unique one-sided null relation and proves, over the stored exact
cyclotomic q-series range, that the six-dimensional bi-Jacobi kernel is exactly
the symmetric-square kernel generated by that one relation.

It then replaces the 21 shared coefficients by 15 observable quotient
coordinates. The two parked sectors have full rank `15/15` on this quotient.


also saves the 15 x 21 quotient matrix needed by the degree-three transport
bootstrap.

## Scope

The relation is verified exactly over every coefficient stored by Cell 32 for
the canonical gamma/3 candidate. Calling it an all-orders q-series identity
requires a separate modular/Sturm-bound or closed-form proof.
