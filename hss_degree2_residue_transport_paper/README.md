```text
Rquad + a*rA + b*rB + c*rC
```

```text
alpha_1(U) = alpha_-3 U^-3 + alpha_-2 U^-2 + alpha_-1 U^-1

beta_1(U)  = beta_-3 U^-3 + beta_-2 U^-2 + beta_const
```

